Tài liệu Quy ước đặt tên class

Quy ước đặt tên class thông dụng trong HTML/CSS

Bố cục (Layout)

Class Chú thích
container Khung chứa, giới hạn chiều rộng nội dung
container-fluid Khung chứa rộng toàn màn hình
wrapper Khung bao bọc bên ngoài
layout Bố cục tổng thể của trang
main Vùng nội dung chính
content Khu vực nội dung
content-wrapper Khung bao của vùng nội dung
sidebar Thanh bên (trái/phải)
aside Nội dung phụ bên cạnh
section Phần / khối nội dung lớn
block Khối nội dung
row Hàng (dòng) trong lưới
col Cột trong lưới
grid Lưới bố cục
flex Bố cục dạng Flexbox
inner Khung bên trong
outer Khung bên ngoài

Header & Footer

Class Chú thích
header Đầu trang
header-top Thanh đầu trang (phía trên)
header-bottom Thanh dưới đầu trang

logo Logo
brand Thương hiệu / tên trang
banner Banner quảng cáo / tiêu đề
hero Khu vực nổi bật đầu trang
footer Chân trang
footer-top Phần trên chân trang
footer-bottom Phần dưới chân trang
copyright Dòng bản quyền
back-to-top Nút về đầu trang

Class Chú thích

submenu Menu con
dropdown Menu xổ xuống
dropdown-menu Danh sách menu xổ xuống
dropdown-item Mục trong menu xổ xuống
breadcrumb Đường dẫn phân cấp (vị trí trang)
pagination Phân trang
tabs Bộ thẻ chuyển nội dung
tab-pane Nội dung của một tab
active Mục đang được chọn

Class Chú thích

btn-light Nút màu sáng
btn-dark Nút màu tối

btn-sm Nút nhỏ
btn-lg Nút lớn
btn-block Nút chiếm trọn chiều rộng
btn-icon Nút chỉ có biểu tượng

Class Chú thích

is-invalid Trường không hợp lệ
help-text Dòng hướng dẫn nhỏ

Class Chú thích

Class Chú thích
