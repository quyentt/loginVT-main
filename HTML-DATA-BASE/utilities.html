Object fit

Cách ảnh/video lấp đầy khung mà không méo. Áp lên thẻ <img>.

Class Chú thích
object-fit-coverLấp đầy, cắt bớt phần thừa
object-fit-containHiện trọn ảnh, có thể chừa viền
object-fit-fillKéo giãn lấp đầy (có thể méo)
Code 
<img src="..." class="object-fit-cover" width="120" height="120">

Spacing

Margin (m) và padding (p), giá trị 0–5 (hoặc auto). Cú pháp: {m|p}{cạnh}-{0..5}.

Ký hiệu cạnh Ý nghĩa
t / btop / bottom
s / estart (trái) / end (phải)
x / yngang (trái+phải) / dọc (trên+dưới)
(không)cả 4 cạnh
Code 
<div class="mt-3">   <!-- margin-top -->
<div class="px-2">   <!-- padding ngang -->
<div class="mx-auto"><!-- tự canh giữa -->
<div class="p-0 m-0"><!-- bỏ hết -->

text-nowrapKhông xuống dòng

Vertical align

Canh dọc cho phần tử inline, inline-block hoặc ô bảng.

Code 
<span class="align-middle">...</span>
<!-- align-baseline / align-top / align-middle / align-bottom -->

Visibility

invisible ẩn phần tử nhưng vẫn chiếm chỗ (khác d-none — không chiếm chỗ).

Code 
<div class="visible">...</div>
<div class="invisible">...</div>   <!-- ẩn nhưng giữ chỗ -->

Z-index

Thứ tự lớp chồng. Class tiện ích: z-n1, z-0, z-1, z-2, z-3.

Code 
<div class="position-relative">
    <div class="position-absolute z-1">Trên</div>
    <div class="position-absolute z-0">Dưới</div>
</div>
