Tài liệu Helpers

Helpers - Lớp tiện ích hỗ trợ (Bootstrap)

Clearfix

Xóa float bên trong một phần tử (để phần tử cha bao trọn các con đang float). Thêm class clearfix vào thẻ cha.

Code 
<div class="clearfix">
    <div class="float-start">Trôi trái</div>
    <div class="float-end">Trôi phải</div>
    <!-- cha tự bao trọn 2 con float -->
</div>

Colored links

Giao diện
Primary   Success   Danger   Secondary
Code 
<a href="#" class="link-primary">Primary</a>
<a href="#" class="link-success">Success</a>
<a href="#" class="link-danger">Danger</a>

Ratio

Giữ tỉ lệ khung hình cho video / iframe / ảnh nhúng (responsive). Dùng ratio + ratio-16x9 (hoặc 1x1, 4x3, 21x9).

Giao diện (16x9)
16:9
Code 
<div class="ratio ratio-16x9">
    <iframe src="..."></iframe>
</div>

Position

Đặt vị trí phần tử, hay dùng để gắn badge lên góc.

Giao diện
Class Chú thích
position-relative Định vị tương đối (mốc cho con)
position-absolute Định vị tuyệt đối
top-0 / start-0 / end-0 / bottom-0 Dính về cạnh
top-50 / start-50 Về giữa (50%)
translate-middle Dịch về đúng tâm điểm
fixed-top / sticky-top Cố định / dính trên cùng
Code 
<button class="btn btn-primary position-relative">
    Hộp thư
    <span class="position-absolute top-0 start-100 translate-middle
                 badge rounded-pill bg-danger">9+</span>
</button>

Visually hidden

Ẩn phần tử khỏi màn hình nhưng vẫn để trình đọc màn hình (screen reader) đọc được — hỗ trợ người khiếm thị. Khác với d-none (ẩn hoàn toàn).

Code 
<button class="btn btn-primary">
    <i class="fa-solid fa-trash"></i>
    <span class="visually-hidden">Xóa</span>
</button>

<!-- Hiện lại khi focus (bỏ qua link) -->
<a class="visually-hidden-focusable" href="#main">Bỏ qua tới nội dung</a>

Stretched link

Cho phép bấm vào toàn bộ khối (card) chứ không chỉ chữ link. Thêm stretched-link vào thẻ <a>, và position-relative vào khối cha.

Giao diện (bấm cả thẻ)
Tiêu đề thẻ

Cả thẻ này là một liên kết.

Code 
<div class="card position-relative">
    <div class="card-body">
        <h5 class="card-title">Tiêu đề</h5>
        <a href="#" class="stretched-link"></a>
    </div>
</div>

Text truncation

Cắt bớt văn bản dài trên 1 dòng và thêm dấu …. Dùng text-truncate (cần phần tử có chiều rộng giới hạn).

Giao diện
Đây là một dòng văn bản rất dài sẽ bị cắt bớt và hiển thị dấu ba chấm ở cuối.
Code 
<div class="text-truncate" style="max-width:240px;">
    Đây là một dòng văn bản rất dài sẽ bị cắt bớt...
</div>
