Overview - Giới thiệu

Trang này tập hợp các mẫu biểu mẫu (form) chuẩn của dự án. Mục đích để mọi người khi code copy đúng cấu trúc, giữ giao diện đồng nhất.

Quy ước chung của một trường nhập liệu:

<div class="item item-input">
    <div class="label">Tên trường</div>
    <div class="form-item form-add-info">
        <input class="form-control" placeholder="...">
    </div>
</div>
  • item-input: trường nhập text. item-select: trường chọn (dropdown).
  • .label: nhãn trường. .form-item.form-add-info: bọc ô nhập.
  • Mọi ô nhập dùng class chuẩn form-control / form-select.

Họ và tên
Ghi chú

<!-- Ô nhập text -->
<div class="item item-input">
    <div class="label">Họ và tên</div>
    <div class="form-item form-add-info">
        <input class="form-control" placeholder="Gợi ý: Nguyễn Văn A">
    </div>
</div>

<!-- Ô nhập nhiều dòng -->
<div class="item item-input">
    <div class="label">Ghi chú</div>
    <div class="form-item form-add-info">
        <textarea class="form-control" rows="3" placeholder="Nhập ghi chú..."></textarea>
    </div>
</div>

Select - Ô chọn

Giới tính
Code

Checks & radios - Chọn / Tích

Giao diện
Code 
<!-- Checkbox -->
<div class="form-check">
    <input class="form-check-input" type="checkbox" id="check1">
    <label class="form-check-label" for="check1">Đồng ý điều khoản</label>
</div>

<!-- Radio (cùng name = chọn 1) -->
<div class="form-check">
    <input class="form-check-input" type="radio" name="gt" id="r1">
    <label class="form-check-label" for="r1">Nam</label>
</div>

<!-- Switch (công tắc) -->
<div class="form-check form-switch">
    <input class="form-check-input" type="checkbox" id="sw1">
    <label class="form-check-label" for="sw1">Bật nhận thông báo</label>
</div>

Input group - Ô nhập có tiền / hậu tố

VNĐ
Code

<div class="input-group">
    <span class="input-group-text"><i class="fal fa-search"></i></span>
    <input class="form-control" placeholder="Tìm kiếm...">
</div>

<!-- Ký tự ở đầu -->
<div class="input-group">
    <span class="input-group-text">@</span>
    <input class="form-control" placeholder="Tên đăng nhập">
</div>

<!-- Hậu tố ở cuối -->
<div class="input-group">
    <input class="form-control" placeholder="Số tiền">
    <span class="input-group-text">VNĐ</span>
</div>

Layout - Bố cục form

Form chia cột theo lưới, mỗi trường bọc trong item. Cột phổ biến: col-lg-3 (4 trường/hàng), col-md-6 (2 trường/hàng trên màn vừa).

Ngày sinh

<div class="row aps-form-data">
    <div class="col-12 col-md-6 col-lg-3 item item-input">
        <div class="label">Họ và tên</div>
        <div class="form-item form-add-info">
            <input class="form-control" placeholder="Nguyễn Văn A">
        </div>

    <!-- ... các trường khác lặp lại cấu trúc trên ... -->
</div>

<!-- Dạng nhãn nằm ngang: thêm class label-ngang vào hàng -->
<div class="row aps-form-data label-ngang"> ... </div>

Validation - Kiểm tra hợp lệ

Giao diện
Email (hợp lệ)
Thông tin hợp lệ!
Email (lỗi)
Email không đúng định dạng.
Code 
<!-- Hợp lệ -->
<input class="form-control is-valid" value="a@b.com">
<div class="valid-feedback">Thông tin hợp lệ!</div>

<!-- Lỗi -->
<input class="form-control is-invalid" value="abc">
<div class="invalid-feedback">Email không đúng định dạng.</div>
