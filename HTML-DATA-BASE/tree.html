Danh mục Tree

Các dạng cây (Tree) đang dùng trong hệ thống

Tree Cơ cấu tổ chức (jstree)

Cây đơn vị hành chính dùng plugin jstree (theme jstree-default). Mỗi node là li.jstree-node; node có con thêm jstree-open/jstree-closed và một ul.jstree-children lồng bên trong. Bọc cả cây trong .trees-coCau-toChuc để nhóm nút thao tác (Chi tiết / Sửa / Xóa / Thêm) tự hiện khi rê chuột vào dòng.

Việc đóng/mở node do plugin jstree xử lý bằng JS. Trang demo này chỉ hiển thị cấu trúc & giao diện tĩnh.

<div class="trees-coCau-toChuc">
    <div class="jstree jstree-default" role="tree">

            <li role="treeitem" class="jstree-node jstree-open treeitem d-flex">
                <i class="jstree-icon jstree-ocl"></i>

                    <div class="list-btn treeitem-btn aps-gap-10">
                        <a class="btn btn-outline-info">Chi tiết</a> ...
                    </div>
                </div>
                <ul role="group" class="jstree-children">
                    <li class="jstree-node jstree-leaf treeitem-2">...</li>
                </ul>
            </li>
        </ul>
    </div>
</div>

Tree Phân quyền / Danh mục dữ liệu (jstree động)

Các màn Phân quyền vai trò, Danh mục dữ liệu, Quản trị quyền dữ liệu… dùng một <div> rỗng làm vùng chứa (zone), jstree sẽ tự đổ dữ liệu và sinh markup khi trang tải. Bạn chỉ cần đặt đúng id/class vùng chứa.

Vùng chứa (class / id) Dùng ở màn
.aps-vaitro-cn-l · #zone_vaitroungdung Phân quyền chức năng theo vai trò
.aps-vaitro-nguoidung-l Vai trò người dùng
.aps-function-l Cây chức năng ứng dụng
#zone_danhmuctenbang_dmdl Danh mục dữ liệu (tên bảng)
Code (vùng chứa rỗng) 
<div class="col-sm-3 aps-box-l aps-vaitro-cn-l">
    <!-- jstree tự render cây vào trong div này -->
    <div id="zone_vaitroungdung" class="jstree jstree-default" role="tree"></div>
</div>

Menu điều hướng dạng cây (sidebar accordion)

Menu trái của hệ thống cũng là một dạng cây 2 cấp, dựng bằng .sidebar-menu + Bootstrap collapse. Mục cha là .sidebar-menu-header với data-bs-toggle="collapse", nhóm con nằm trong .sidebar-menu-sub.collapse.

Code (rút gọn) 
<div class="accordion" id="sidebar-menu">
    <div class="sidebar-menu-item">
        <a class="sidebar-menu-header collapsed" data-bs-toggle="collapse"
            data-bs-target="#sbForms" href="javascrip:void(0)">
            <i class="fa-light fa-file-user item-icon"></i>
            <span>Forms</span>
            <i class="fal fa-chevron-right item-arrow-icon"></i>
            <i class="fal fa-chevron-down item-arrow-icon down"></i>
        </a>
        <div class="sidebar-menu-sub collapse" id="sbForms" data-bs-parent="#sidebar-menu">
            <div class="sidebar-menu-sub-content">
                <a href="forms.html#overview">Overview</a>
                <a href="forms.html#select">Select</a>
            </div>
        </div>
    </div>
</div>

Xem trực tiếp ở menu bên trái của trang này.

Danh mục Tree với Bootstrap

Tree bằng Bootstrap

Bootstrap 5 không có component tree riêng. Cách dựng phổ biến: lồng các danh sách và dùng collapse để đóng/mở nhánh, kèm icon chevron xoay. Demo dưới đây bấm mở/đóng được ngay.

Code (rút gọn) 
<ul class="tree-bs">
    <li class="tree-bs-item">
        <a class="tree-bs-toggle" data-bs-toggle="collapse"
            href="#bsKhoaCNTT" role="button" aria-expanded="true">
            <i class="fa-solid fa-chevron-down tree-bs-caret"></i>
            <i class="fa-light fa-folder-tree"></i> Khoa CNTT

        <ul class="collapse show" id="bsKhoaCNTT">
            <li class="tree-bs-item">
                <span class="tree-bs-leaf">Lớp CNTT-K66</span>
            </li>
        </ul>
    </li>
</ul>

Class & ý nghĩa

Class / Thuộc tính Ý nghĩa
1. Tree Bootstrap (dựng tay)
data-bs-toggle="collapse" Bấm để mở/đóng một nhánh con.
href="#id" / data-bs-target Trỏ tới nhánh (.collapse) cần đóng/mở.
.collapse Nhánh đang ẩn (mặc định đóng).
.collapse.show Nhánh đang mở sẵn khi tải trang.
.list-group / .list-group-item Có thể dùng thay <ul> để mỗi node là một dòng có viền.
aria-expanded Báo trạng thái mở/đóng cho trợ năng; cũng dùng để xoay chevron.
2. Tree jstree (hệ thống dùng)
.jstree.jstree-default Vùng cây gốc, áp theme mặc định của jstree.
.jstree-container-ul <ul> bao toàn bộ node cấp 1.
.jstree-node Một node (<li>) trong cây.
.jstree-leaf Node lá — không có con.
.jstree-open / .jstree-closed Node cha đang mở / đang đóng.
.jstree-last Node cuối cùng trong nhóm.
.jstree-children <ul> chứa các node con (lồng nhau).
.jstree-ocl Icon mũi tên đóng/mở (open-close-leaf).
.jstree-anchor Thẻ bấm chứa tên node.
.jstree-themeicon Icon đại diện cho node (gắn icon Font Awesome).
3. Class riêng của dự án
.trees-coCau-toChuc Bọc cả cây; bật hiện nút thao tác khi hover dòng.
.tree-list-title Hàng chứa tên node + nhóm nút thao tác.
.treeitem / .treeitem-2 Node cấp cha / node cấp con.
.treeitem-btn / .treeitem-btn-con Nhóm nút (Chi tiết / Sửa / Xóa / Thêm) ẩn, hiện khi hover.
4. Menu cây (sidebar)
.sidebar-menu Khối menu trái dạng cây.
.sidebar-menu-item Một mục menu (cha hoặc đơn).
.sidebar-menu-header Dòng tiêu đề mục cha (bấm để mở nhóm con).
.sidebar-menu-sub Nhóm mục con, dùng kèm .collapse.

Lưu ý khi dùng

  • Tree thật của hệ thống chạy bằng plugin jstree — markup do JS sinh ra, đừng viết tay toàn bộ.
  • Khi cần tree nhẹ, không phụ thuộc plugin: dùng collapse của Bootstrap như demo bên trên.
  • Mỗi nhánh collapse cần id riêng, không trùng nhau trên cùng trang.
  • Chevron xoay bằng CSS dựa vào class .collapsed Bootstrap tự thêm/bỏ trên nút bấm.
