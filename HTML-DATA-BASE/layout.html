Tài liệu Layout

Layout - Hệ thống bố cục (Bootstrap Grid)

Breakpoints - Điểm ngắt màn hình

Bootstrap chia màn hình thành 6 mốc. Hậu tố (infix) gắn vào class để áp dụng theo từng cỡ màn hình, ví dụ col-md-6.

Breakpoint Hậu tố Kích thước
Extra small—< 576px
Smallsm≥ 576px
Mediummd≥ 768px
Largelg≥ 992px
Extra largexl≥ 1200px
XX largexxl≥ 1400px

Containers - Khung chứa

Khung chứa bọc nội dung và canh giữa. 3 loại chính:

Class Chú thích
containerRộng cố định, đổi theo từng breakpoint
container-fluidRộng 100% mọi màn hình
container-md100% đến md, sau đó cố định
Code 
<div class="container">
    <!-- nội dung canh giữa, rộng cố định -->
</div>

<div class="container-fluid">
    <!-- rộng tràn 100% -->
</div>

Grid - Lưới 12 cột

Lưới gồm .container > .row > .col. Mỗi hàng chia tối đa 12 cột.

Giao diện
.col
.col
.col
.col-8
.col-4
Code 
<div class="container">
    <div class="row">
        <div class="col">.col</div>
        <div class="col">.col</div>
        <div class="col">.col</div>
    </div>
    <div class="row">
        <div class="col-8">.col-8</div>
        <div class="col-4">.col-4</div>
    </div>
</div>

Columns - Cột

Giao diện (cột responsive)
col-lg-3
col-lg-3
col-lg-3
col-lg-3
col-auto
col (phần còn lại)
Code 
<!-- Responsive: 1 cột (mobile) → 2 cột (md) → 4 cột (lg) -->
<div class="row">
    <div class="col-12 col-md-6 col-lg-3">...</div>
    <div class="col-12 col-md-6 col-lg-3">...</div>
</div>

<!-- col-auto: rộng theo nội dung; col: chiếm phần còn lại -->
<div class="row">
    <div class="col-auto">...</div>
    <div class="col">...</div>
</div>

Canh / lệch cột: offset-md-2 (đẩy sang phải), align-items-center, justify-content-between, order-1.

Gutters - Khoảng cách cột

Gutter là khoảng cách giữa các cột. Dùng class g-* (cả 2 chiều), gx-* (ngang), gy-* (dọc), giá trị 0–5.

Giao diện (g-3)
col-6
col-6
col-6
col-6
Code 
<div class="row g-3">      <!-- gutter 2 chiều -->
<div class="row gx-5">     <!-- chỉ khoảng cách ngang -->
<div class="row gy-2">     <!-- chỉ khoảng cách dọc -->
<div class="row g-0">      <!-- bỏ hết khoảng cách -->

<!-- Tùy chỉnh trực tiếp -->
<div class="row" style="--bs-gutter-x: 20px;">

Giao diện
Trái Giữa Phải
Class Chú thích
d-flexBật flexbox

Z-index - Thứ tự lớp chồng

Quy định phần tử nào nằm trên khi chồng lên nhau. Bootstrap có sẵn class tiện ích z-* và z-index cho component.

Class / Giá trị Chú thích
z-n1z-index: -1 (nằm dưới)
z-0z-index: 0
z-1 / z-2 / z-3Tăng dần lớp chồng
dropdown (1000)z-index của menu xổ
fixed / sticky (1030)Thanh cố định
modal (1055)Hộp thoại
tooltip (1080)Chú thích nổi trên cùng
Code 
<div class="position-relative">
    <div class="position-absolute z-1">Lớp trên</div>
    <div class="position-absolute z-0">Lớp dưới</div>
</div>
