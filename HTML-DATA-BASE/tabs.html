Tabs cơ bản

Dùng .nav.nav-tabs cho thanh tiêu đề và .tab-content chứa các .tab-pane. Mỗi nút mở tab dùng data-bs-toggle="tab" và data-bs-target trỏ tới id của pane tương ứng.

Giao diện
Sinh viên Nguyễn Văn An — Lớp CNTT-K65.
Điểm trung bình tích lũy: 8.2 — Xếp loại Giỏi.
Học phí học kỳ này: Đã hoàn thành.
Code 
<ul class="nav nav-tabs" role="tablist">
    <li class="nav-item" role="presentation">
        <button class="nav-link active" data-bs-toggle="tab"
            data-bs-target="#tb-thongtin" type="button" role="tab">Thông tin</button>
    </li>
    <li class="nav-item" role="presentation">
        <button class="nav-link" data-bs-toggle="tab"
            data-bs-target="#tb-diem" type="button" role="tab">Điểm</button>
    </li>
</ul>
<div class="tab-content pt-10">
    <div class="tab-pane fade active show" id="tb-thongtin" role="tabpanel">...</div>
    <div class="tab-pane fade" id="tb-diem" role="tabpanel">...</div>
</div>

Tabs dạng Pills

Thay .nav-tabs bằng .nav-pills và data-bs-toggle="pill" để có kiểu nút bo tròn, không viền dưới.

Giao diện
Danh sách tất cả sinh viên trong khoa.
Sinh viên đang theo học.
Sinh viên đã tốt nghiệp.

    <li class="nav-item" role="presentation">
        <button class="nav-link active" data-bs-toggle="pill"
            data-bs-target="#pl-all" type="button" role="tab">Tất cả</button>
    </li>
</ul>
<div class="tab-content">
    <div class="tab-pane fade active show" id="pl-all" role="tabpanel">...</div>
</div>

Tabs có viền & icon (kiểu dùng trong dự án)

Bọc khối tabs trong .aps-tabs-border để có khung viền bo, kèm khoảng đệm .pl20 .pr20 .pb-20 .pt10 như các màn hồ sơ sinh viên đang dùng. Thêm icon vào trong từng nút để dễ nhận diện.

Căn chỉnh & chia đều

Thêm class tiện ích của Bootstrap vào <ul> để đổi cách dàn các tab.

Class (đặt trên ul.nav) Tác dụng
.nav-fill Các tab chiếm hết chiều rộng theo độ rộng nội dung từng tab.
.nav-justified Mọi tab có chiều rộng bằng nhau, lấp đầy hàng.
.justify-content-center Dồn nhóm tab ra giữa.
.justify-content-end Dồn nhóm tab về bên phải.
.flex-column Xếp tab theo chiều dọc (tab dạng cột bên trái).

<ul class="nav nav-tabs nav-fill" role="tablist"> ... </ul>
<ul class="nav nav-pills justify-content-end" role="tablist"> ... </ul>

Tabs của Bootstrap

Class và ý nghĩa

Class / Thuộc tính Ý nghĩa
1. Thanh tiêu đề tab
.nav Lớp gốc bắt buộc cho danh sách điều hướng/tab.
.nav-tabs Kiểu tab có viền, gạch chân tab đang chọn.
.nav-pills Kiểu nút bo tròn (pill), nền màu khi active.
.nav-item Mỗi mục tab, thường là <li>.
.nav-link Nút/đường dẫn bấm để mở tab (đặt trên <button> hoặc <a>).
.nav-link.active Đánh dấu tab đang được chọn.
.disabled Vô hiệu hóa một tab (không bấm được).
2. Vùng nội dung
.tab-content Khối bao tất cả các pane nội dung.
.tab-pane Một vùng nội dung tương ứng với một tab.
.fade Thêm hiệu ứng mờ dần khi chuyển tab.
.active.show Pane đang hiển thị (cần cả 2 khi dùng .fade).
3. Thuộc tính kích hoạt
data-bs-toggle="tab" Kích hoạt chuyển tab (dùng với .nav-tabs).
data-bs-toggle="pill" Kích hoạt chuyển tab (dùng với .nav-pills).
data-bs-target="#id" Trỏ tới id của .tab-pane sẽ mở.
role="tablist" / "tab" / "tabpanel" Vai trò ARIA giúp trợ năng (accessibility).
aria-selected Báo tab nào đang được chọn cho trình đọc màn hình.
4. Căn chỉnh & bố cục
.nav-fill Tab giãn lấp đầy theo nội dung.
.nav-justified Mọi tab rộng bằng nhau.
.flex-column Xếp tab theo chiều dọc (cột bên trái).
5. Class riêng của dự án
.aps-tabs-border Bọc khung viền bo quanh khối tabs (dùng ở màn hồ sơ SV).
.aps-tabs / .aps-tabs-blue Bộ tab kiểu cũ (AdminLTE) với màu chủ đạo.
.hosodauvao-tabs Biến thể tab dùng cho khối hồ sơ đầu vào.

Lưu ý khi dùng

  • Mỗi .tab-pane phải có id trùng với data-bs-target của nút mở tab.
  • Tab và pane mặc định cần class active show; các pane còn lại chỉ để fade.
  • Khi có nhiều khối tabs trên cùng trang, đặt id khác nhau để tránh xung đột.
  • Tabs hoạt động nhờ bootstrap.bundle.js — nhớ nhúng file JS này ở cuối trang.
